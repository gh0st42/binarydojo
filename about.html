About Binary Dojo

Home

Binary Dojo is a collection of web-based tools for my own personal use and potentially other (experienced) martial artists. It is a work in progress, and new tools will be added over time. The tools are designed to be flexible and accessible from any device with a web browser.

The projects does not teach you karate, but it can give you inspiration for solo training if you already know karate. Use at your own risk!

This is a personal project of mine which started as a visual basic program on my desktop years ago, then was ported to swift for an iOS app and now finally was made available to all devices via a simple web application.

Feel free to contribute to the Binary Dojo project via GitHub pull requests or by opening issues there. Just remember this is a just-for-fun side project. Code wise, I try to keep everything simple, no nodejs, just plain html, css and javascript, and minimal to no external dependencies - all public domain or open source. It should be useable without a browser or internet connection if you open the files locally.

Currently, the only external dependency is the awesome simplecss.
